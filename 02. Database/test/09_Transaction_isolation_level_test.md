## Transaction Isolation Level  

1.	다음 중 대부분 DBMS가 채택하고 있는 기본 트랜잭션 격리성 수준(Transaction Isolation Level)인 것은?  

     A.	Read Uncommitted  
  
     B.	Read Committed  
  
     C.	Repeatable Read  
  
     D.	Serializable  
     
     <br>
  
2.	O, X 문제  

     A.	트랜잭션 격리성 수준(Transaction Isolation Level)을 Serializable로 상향 조정하면 일반적으로 동시성과 일반성이 같이 높아진다. (O, X)
